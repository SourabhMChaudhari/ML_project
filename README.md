# ML_project
[ITCS  6156 ] Group 2 : Sales Analysis and Forecasting

We have 4 files:

1. Visualization
This files includes visualization and preprocessing  for the dataset. It includes visualization of quantity sales, profits and per different US states and regions.

2. Predict sales and profits
This files includes prediction of quantity sold using Linear regression that uses MAPE.

3. Profit using Neural Nets
This files includes predicted profits using Deep Neural Network.

4. review Sentiment Analysis
This files includes Sentimental Analysis of the Customer reviews and feedback and predictions for the product reaction for new product.


Dataset used: https://cseweb.ucsd.edu/~jmcauley/datasets/amazon_v2/
